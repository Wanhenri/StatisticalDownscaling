## Required Dependencies

Required dependencies for RFSD package are listed below.

### Python Packages

* gdal 1.10.1
* matplotlib 1.4.2 (basemap toolkit 1.0.7)
* netCDF4 1.1.0 (netCDF 4.2.1.1, HDF5 1.8.12)
* numpy 1.9.1
* rpy2 2.3.9 (R version 3.0.3)
* scipy 0.13.3

### R Packages

* gstat 1.0-18
* sp 1.0-14
